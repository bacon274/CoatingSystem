# KY040 rotary decoder library
KY-040 rotary encoder library for arduino and esp8266 devices.

## Features

- Callback functions on click and rotate events
- Software debouncing
- Interrupts support

## Tested on hardware

- Arduino Uno
- Wemos D1 R2

## Wiring

The wiring for ESP8266 can be as follows :

- Vcc     --> 3.3V
- GND     --> Ground
- SW      --> D7
- DT      --> D6
- CLK     --> D5
